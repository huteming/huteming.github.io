```js
function render(template, data) {
  return template.replace(/\{\{([^\}]+)\}\}/g, (str, $1) => {
    return data[$1]
  })
}
```

测试代码

```js
const template = '我是{{name}}，年龄{{age}}，性别{{sex}}'
const data = {
  name: '姓名',
  age: 18,
}
console.log(render(template, data))
```
