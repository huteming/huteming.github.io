## 默认值

```css
.flex-item {
  /* flex: 0 1 auto; */
}
```

## auto

```css
.flex-item {
  /* flex: auto; */
  flex: 1 1 auto;
}
```

## none

```css
.flex-item {
  /* flex: none; */
  flex: 0 0 auto;
}
```

## 非负数字

```css
.flex-item {
  /* flex: 6; */
  flex: 6 1 0%;
}
```

## 长度或百分比

```css
.flex-item {
  /* flex: 10%; */
  flex: 1 1 10%;
}
```
