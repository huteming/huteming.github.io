## 箭头函数与普通函数的区别

### 不会创建自己的 this

箭头函数不会创建自己的 this，它只会`从自己的作用域链的上一层继承 this`。

### 继承而来的 this 指向永远不变

由于 箭头函数没有自己的 this 指针，通过 call() 或 apply() 方法调用一个函数时，只能传递参数，他们的第一个参数会被忽略。

### 不能作为构造函数使用

### 不绑定 arguments

箭头函数没有自己的 arguments 对象。在箭头函数中访问 arguments 实际上获得的是外层局部（函数）执行环境中的值。
